致富经报道：19岁的她靠棋牌游戏赚取到人生第一桶金

2019年3月1日，家住重庆大渡口某小区的陈丽阿姨发现自己正在上大学的孩子银行卡里居然有20多万！！在父母的逼问下，她终于说出这些钱都是玩一款名叫“魔盒娱乐”的棋牌游戏赚来的...

“在学校寝室里面玩游戏，朋友让我玩一个叫“魔盒娱乐”的游戏，说是里面的捕鱼很好玩，还能赚钱。然后我推脱不过就抱着试试的心态下载下来充了200块钱，没想到一把打中一条金龙，一下子返了我2000多。接着就把赢来的钱又去学着玩了一下龙虎斗，没想到运气很好，老是中奖了。然后就这样悄悄玩了两个月，就有了这么多。”陈阿姨的孩子对着记者这样说道。

小娜还积极的分享了未来的计划，准备用游戏里赚来的20多万尝试着做点小生意，并且小娜也愿意把这款游戏分享给致富经的读者们，但是小娜告诉记者，希望大家把这当作一个致富机会的同时切勿沉迷，见好就收才能细水长流。
